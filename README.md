# 校园美食店铺可视化推荐与评价小程序

> 基于 Taro 跨端框架开发的校园美食推荐微信小程序

## 项目简介

"今天吃什么？"这是每个学生每天都会面临的灵魂拷问。本项目旨在开发一款校园美食推荐小程序，为学生提供便捷的美食发现工具。

## 功能特性

- **首页推荐**：随机推荐、今日推荐、热门榜单、新店推荐
- **店铺浏览**：区域筛选、类型筛选、关键词搜索
- **店铺详情**：店铺信息、菜品列表、评价列表
- **评价互动**：发表评价、点赞评价、图片上传
- **收藏管理**：收藏店铺、收藏菜品
- **UGC贡献**：上传新店铺、上传新菜品
- **用户中心**：登录管理、个人信息、收藏管理

## 技术栈

### 前端
- **框架**：Taro 4.1.9
- **UI库**：React 18.x
- **开发语言**：TypeScript 5.x
- **样式**：SCSS
- **UI组件**：Vant Weapp

### 后端（计划中）
- **运行时**：Node.js 18.x
- **框架**：Express 4.18.x
- **数据库**：MySQL 8.0
- **认证**：JWT

## 快速开始

### 环境要求

- Node.js >= 16.x
- npm >= 8.x
- 微信开发者工具

### 安装依赖

```bash
# 安装前端依赖
cd taro-mini-app
npm install

# 安装后端依赖（可选）
cd ../backend
npm install
```

### 开发运行

```bash
# 前端开发
cd taro-mini-app
npm run dev:weapp

# 后端开发（可选）
cd backend
npm run dev
```

### 项目构建

```bash
# 构建微信小程序
npm run build:weapp
```
